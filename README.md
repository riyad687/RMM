# RMM Instagram Cookie Bot

এই বটটি দিয়ে আপনি খুব সহজে ইনস্টাগ্রাম আইডি থেকে কুকি এক্সট্র্যাক্ট করতে পারবেন।

### যেভাবে ব্যবহার করবেন:
১. টার্মাক্স (Termux) ওপেন করে এই কমান্ড দিন:
   `git clone https://github.com/riyad687/RMM`

২. ফোল্ডারে প্রবেশ করুন:
   `cd RMM`

৩. প্রয়োজনীয় টুলস ইন্সটল করুন:
   `pip install -r requirements.txt`

৪. বট চালু করুন:
   `python main.py`

*বট রান করার পর আপনার টেলিগ্রাম বট টোকেনটি দিন এবং টেলিগ্রামে এক্সেল ফাইল পাঠান।*
